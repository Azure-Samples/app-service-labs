# Welcome to App Service Labs

**App Service Labs** is a collection of hands-on tutorials and workshops for
building, deploying, and operating web apps, APIs, and containers on
[Azure App Service](https://azure.microsoft.com/products/app-service/).

<UnderConstruction title="Content is under construction">
  We're actively building out these labs for Azure App Service. Workshops and
  getting-started guides are on the way — check back soon, and follow the
  [App Service blog](https://azure.github.io/AppService/) for updates.
</UnderConstruction>

## What you'll find here

Once the labs are ready, you'll be able to work through practical,
self-paced tutorials covering topics such as:

- **Getting started** — deploy your first web app, API, or container (.NET, Node.js, Python, Java, PHP)
- **Scenarios & modern apps** — host MCP servers, AI apps, and OAuth-secured APIs on App Service
- **Deployment & CI/CD** — GitHub Actions, Azure DevOps, and deployment slots
- **Configuration, networking, security & identity**
- **Scaling, performance, monitoring & diagnostics**
- **Data & integration, reliability & operations**
- **Migration & modernization** — including legacy Windows apps on Managed Instance

Every lab is hands-on and reproducible, shows Windows and Linux, and covers
the Azure Developer CLI (`azd`), Azure CLI, and the portal. Labs complement the
[official documentation](https://learn.microsoft.com/azure/app-service/) — they
don't replace it — and link back to the reference for depth.

## Contribute
