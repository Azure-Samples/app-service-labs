# Welcome to App Service Labs

**App Service Labs** is a collection of hands-on tutorials and workshops for
building, deploying, and operating web apps, APIs, and containers on
[Azure App Service](https://azure.microsoft.com/products/app-service/).

Every lab is hands-on and reproducible. Most show the same path three ways -
the Azure Developer CLI (`azd`), the Azure CLI, and the Azure portal - so you
can follow along whichever way you prefer. Labs complement the
[official documentation](https://learn.microsoft.com/azure/app-service/); they
don't replace it, and each one links back to the reference for depth.

## Follow a learning path

Prefer a guided journey? The [enterprise web app learning path](/docs/learning-paths/overview)
carries a single app - Zava Widgets - from a plain first deploy to an
enterprise-grade app, adding one capability per step: configuration, a
passwordless database, secrets, resilience, monitoring, private networking, and
automated delivery.

## Available labs

Prefer to pick a single topic? Start with the [Getting started](/docs/getting-started/overview)
track, then explore scenario and platform labs:

- **Getting started** - [Deploy your first web app](/docs/getting-started/deploy-your-first-web-app)
  (.NET, Node.js, Python, Java, or PHP, on Linux or Windows) and
  [Deploy a custom container](/docs/getting-started/deploy-a-custom-container).
- **Scenarios & modern apps** - [Host an MCP server secured with OAuth](/docs/scenarios-and-modern-apps/host-an-mcp-server)
  and [Build and deploy an AI-powered app](/docs/scenarios-and-modern-apps/deploy-an-ai-app).
- **Deployment & CI/CD** - [Deploy with GitHub Actions](/docs/deployment-and-cicd/deploy-with-github-actions)
  and [Use deployment slots for zero-downtime releases](/docs/deployment-and-cicd/deployment-slots).
- **Scaling & performance** - [Scale your app up and out with autoscale](/docs/scaling-and-performance/autoscale).
- **Monitoring & diagnostics** - [Monitor your app with Application Insights](/docs/monitoring-and-diagnostics/application-insights).
- **Security & identity** - [Reference Key Vault secrets from App Service](/docs/security-and-identity/key-vault-references).
